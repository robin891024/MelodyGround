#  MelodyGround

一個基於 Web 的音樂創作平台，讓使用者可以透過虛擬鍵盤進行即時演奏、錄製和管理音樂作品。

##  功能特色

###  虛擬鍵盤演奏
- **三排按鍵設計**：支援三個八度的音符演奏
  - `Q-W-E-R-T-Y-U`：高一個八度 (C-D-E-F-G-A-B)
  - `A-S-D-F-G-H-J`：當前八度 (C-D-E-F-G-A-B)
  - `Z-X-C-V-B-N-M`：低一個八度 (C-D-E-F-G-A-B)
- **即時音訊回饋**：使用 Tone.js 提供高品質音訊輸出
- **視覺化回饋**：按鍵按下時的即時視覺效果

###  多樂器支援
- 鋼琴 (Piano)
- 吉他 (Guitar)
- 合成器

###  音樂控制
- **音域調整**：支援 5 個八度範圍 (低音 2 - 高音 6)
- **節奏控制**：可調整播放速度 (60-200 BPM)
- **音量控制**：主音量調節 (0-100%)

###  錄製與播放
- **即時錄製**：記錄演奏的每個音符和時間點
- **播放功能**：重播已錄製的作品
- **音符計數**：即時顯示錄製的音符數量

###  作品管理
- **雲端儲存**：將作品儲存到後端資料庫
- **作品列表**：瀏覽和管理所有已儲存的作品
- **MIDI 匯出**：將作品匯出為標準 MIDI 格式

###  使用者系統
- **註冊/登入**：安全的使用者認證系統
- **JWT 驗證**：基於 Token 的身份驗證
- **個人作品庫**：每位使用者擁有獨立的作品空間
